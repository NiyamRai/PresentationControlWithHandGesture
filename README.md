# PresentationControlWithHandGesture

INSTALL FOLLOWING PYTHON LIBRARIES 

OpenCV:
pip install opencv-python

MEDIAPIPE:
pip install mediapipe

KEYBOARD:
pip install keyboard

Keep your palm 25-50 cm away from the camera for better results.
